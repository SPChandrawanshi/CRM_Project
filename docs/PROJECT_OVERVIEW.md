# Project Overview - WhatsApp Multi-Channel CRM

## Vision
Build an enterprise-level Multi-Channel WhatsApp CRM designed for high-performance lead management, qualification, and customer support. The system focuses on a premium SaaS aesthetic with a rigid layout structure to ensure professional standards.

## Core Objectives
- **Enterprise-Grade CRM**: Centralized management of WhatsApp communications.
- **Multi-Channel Support**: Seamless integration across various communication streams.
- **Automated Qualification**: AI-driven lead scoring and qualification.
- **Role-Based Efficiency**: Tailored dashboards for Super Admins, Admins, Managers, Team Leaders, Counselors, and Support.

## Tech Stack
- **Frontend**: React (Functional Components), Vite.
- **Styling**: TailwindCSS (Utility-first CSS).
- **Data Fetching**: React Query (Caching & Synchronization).
- **State Management**: Zustand (Global Store).
- **Animations**: Framer Motion (Subtle micro-interactions).
- **Backend**: Node.js, Express.
- **ORM**: Prisma (Database Management).
- **Real-time**: Socket.io (Live updates).

## Layout & UI Standards
- **Master Layout**: Top Navbar + Left Sidebar (Collapsible) + Main Content.
- **Design Language**: Professional, clean, Indigo-based (#4F46E5).
- **Restrictions**: 
  - No charts.
  - No gradients.
  - No glassmorphism.
  - No heavy animations.
  - Standard enterprise SaaS design only.
